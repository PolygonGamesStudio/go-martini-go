go-martini-go
=============

**To run server type:**

**Setting $GOPATH**
```
$ export GOPATH=$HOME/go

$ export PATH=$PATH:$GOPATH/bin
```

**Download dependens**
```
$ go get github.com/codegangsta/martini
$ go get github.com/coopernurse/gorp
```

**run**
```
$ go run src/server.go src/view.go
```
